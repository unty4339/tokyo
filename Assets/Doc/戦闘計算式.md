
## **1\. 基礎パラメータ（三値）**

モンスターのステータスは、主に以下の3つの隠しパラメータによって決定される。

### **1.1 種族値 (Base Stats)**

そのモンスターの種類ごとに設定された基礎能力値。

* **概要:** 「特定のモモンスターはHPが高い」といった種ごとの特徴。  
* **範囲:** 基本的に各ステータス 1 ～ 255 の範囲。  
* **全6ステータス:** HP(H)、攻撃(A)、防御(B)、特攻(C)、特防(D)、素早さ(S)。

### **1.2 個体値 (Individual Values / IVs)**

モンスター一匹ごとに設定された才能の差。

* **概要:** 同じ種類のモンスターでも、「生まれつき攻撃が高い」個体などが存在する。  
* **範囲:** 0 ～ 31 （32進数で管理されるため）。31が最大で「V」と通称される。  
* **変動:** 生まれた瞬間に決定し、基本的に変わらない

### **1.3 努力値 (Effort Values / EVs)**

バトルやアイテムによって後天的に蓄積される経験値のようなパラメータ。

* **概要:** プレイヤーの育成によって伸ばしたい能力を強化できる（基礎ポイント）。  
* **制限:**  
  * 1つのステータスにつき最大 252 まで（※第6世代以前は255だが、計算上意味があるのは252まで）。  
  * 全ステータスの合計で最大 510 まで。  
* **効果:** レベル100時において、努力値 4 につき実数値が 1 上昇する。

## **2\. ステータス計算式**

最終的なステータス（実数値）は、以下の式で求められる。  
※計算の過程で発生する小数は、その都度切り捨てられる。  
**変数の定義:**

* $Base$: 種族値  
* $IV$: 個体値 (0\~31)  
* $EV$: 努力値 (0\~252)  
* $Lv$: レベル (1\~100)  
* $Nature$: 性格補正 (1.1, 1.0, 0.9)

### **2.1 HP (Hit Point) 計算式**

$$HP \= \\left\\lfloor \\frac{(Base \\times 2 \+ IV \+ \\lfloor \\frac{EV}{4} \\rfloor) \\times Lv}{100} \\right\\rfloor \+ Lv \+ 10$$

### **2.2 HP以外のステータス (A, B, C, D, S) 計算式**

$$Stat \= \\left\\lfloor \\left( \\left\\lfloor \\frac{(Base \\times 2 \+ IV \+ \\lfloor \\frac{EV}{4} \\rfloor) \\times Lv}{100} \\right\\rfloor \+ 5 \\right) \\times Nature \\right\\rfloor$$

### **2.3 性格補正 (Nature)**

性格によって、特定のステータスに以下の補正がかかる（HPにはかからない）。

* **上昇補正:** $\\times 1.1$ (例: いじっぱりなら攻撃1.1倍、特攻0.9倍)  
* **下降補正:** $\\times 0.9$  
* **無補正:** $\\times 1.0$

## **3\. ダメージ計算式**

攻撃技が命中した際のダメージは、以下の順序で計算される。  
基本的に「掛け算をするたびに小数を切り捨てる」処理が行われる。

### **3.1 基本ダメージの計算**

$$Damage\_{Base} \= \\left\\lfloor \\frac{\\lfloor \\frac{Lv \\times 2}{5} \+ 2 \\rfloor \\times Power \\times \\frac{Atk}{Def}}{50} \+ 2 \\right\\rfloor$$  
**変数の定義:**

* $Lv$: 攻撃側のレベル  
* $Power$: 技の威力（効果抜群などで変動する場合あり）  
* $Atk$: 攻撃側の能力値（物理技なら「攻撃」、特殊技なら「特攻」）  
  * ※ランク補正（つるぎのまい等）を含む  
* $Def$: 防御側の能力値（物理技なら「防御」、特殊技なら「特防」）  
  * ※ランク補正（てっぺき等）、リフレクター/ひかりのかべ補正を含む

### **3.2 最終ダメージへの補正適用**

基本ダメージ算出後、以下の要素を順に掛け合わせる（都度切り捨て）。

$$Damage\_{Final} \= Damage\_{Base} \\times Multipliers \\dots$$  

## **4\. 補足：ステータスランク補正**

戦闘中、バフ・デバフで変化する倍率は以下の通り。  
（命中・回避を除く、A/B/C/D/Sの5能力）

| ランク | 倍率 | 分数表現 |
| :---- | :---- | :---- |
| \+6 (最大) | 4.0倍 | 8/2 |
| \+5 | 3.5倍 | 7/2 |
| \+4 | 3.0倍 | 6/2 |
| \+3 | 2.5倍 | 5/2 |
| \+2 (ぐーんと) | 2.0倍 | 4/2 |
| \+1 (あがった) | 1.5倍 | 3/2 |
| **0 (通常)** | **1.0倍** | **2/2** |
| \-1 (さがった) | 0.67倍 | 2/3 |
| \-2 (がくっと) | 0.5倍 | 2/4 |
| ... | ... | ... |
| \-6 (最小) | 0.25倍 | 2/8 |

* 命中率・回避率は分母・分子が 3 基準となる（例：+1で 4/3倍、-1で 3/4倍）。
